# Min-Max Finding Using Divide and Conquer

## Project Overview

This project finds the minimum and maximum elements in an array using two different approaches:

1. Divide and Conquer
2. Naive Approach

The project also counts the number of comparisons performed by each method.

A Streamlit web application is provided to allow users to enter an array and view the results interactively.

---

## Objectives

- Find the minimum element in an array.
- Find the maximum element in an array.
- Implement the Divide and Conquer technique.
- Implement the Naive approach.
- Count the number of comparisons.
- Compare the efficiency of both approaches.
